# Bun

[bun](https://bun.sh/) installer

## Example Usage

```json
"features": {
    "ghcr.io/scaryrawr/dev-feats/bun:1": {
        "version": "1.3.0"
    }
}
```

## Options

| Options Id | Description                   | Type   | Default Value |
| ---------- | ----------------------------- | ------ | ------------- |
| version    | The version of bun to install | string |               |
